MRISA

12 Mar 2014

MRISA (Meta Reverse Image Search Api) is a RESTful api which takes an image url, does a reverse image search and returns a json array with standard search information such as title, description, destination url and the total number of search results. MRISA currently uses the google reverse image search.

Getting Stated

Install the neccessary dependancies using pip

pip install flask pycurl beautifulsoup4

Running the server

python mrisa/mrisa_server.py

Using cURL to make a call to the API

curl -X POST -H "Content-Type: application/json" -d '{"image_url":"http://upload.wikimedia.org/wikipedia/commons/2/29/Voyager_spacecraft.jpg"}' http://localhost:5000/search

There is a client implentation in python in the repository but it should be nice and easy to create one in most languages.
